# Crack the Code League

**Crack the safe code using 5 cryptic clues.  
A pure logic puzzle against the clock.**

Crack the Code League is a competitive logic puzzle game where players race against time to solve a hidden safe code using deduction and reasoning. Every puzzle is fair, skill-based, and designed to challenge your mind.

---

## How It Works
- Each game presents **5 cryptic clues**
- Use pure logic to deduce the correct code
- Faster solutions earn better scores
- Scores determine your position in the **global leaderboard**

---

### How Rankings Work
Your leaderboard position is determined by:
- Your **average solving time** from recent games  
- **Lower average time ranks higher**
- All players are ranked using the **same rules**

---

### League Score Calculation
- Your **League Score** is the average time of your **last 10 completed games**
- Only **completed games** are counted
- Poor results fade as you play more games
- Each new game continuously updates your score

---

### League Entry Requirement
To appear on the leaderboard, you must:
- Complete **at least 25 games**

This ensures fair competition and removes one-time lucky results.

---

### Activity Requirement
To stay visible on the leaderboard:
- Complete **at least 1 game within the last 7 days**

Inactive players are temporarily hidden:
- Scores are **preserved**
- Players **reappear immediately** after playing again

---

### Penalties & Fair Play
- Leaving a game without solving applies a **penalty time**
- Penalties increase your **average score**
- Penalty games **do not increase** your game count
- Penalties fade as you complete more games

---

### Important Notes
- Only **completed games** affect play count
- Only the **last 10 games** influence League Score
- Playing more games makes your score **more accurate**
